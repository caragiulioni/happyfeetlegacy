# Happy Feet Game

A browser-based arcade game where you play as a penguin dodging falling ice blocks in a snowstorm.

## How to Play

- **Desktop:** Use ← and → arrow keys to move
- **Mobile/Tablet:** Tap left or right side of the screen to move

Survive as long as you can and beat your highscore! The longer you last, the harder it gets — the blizzard intensifies, making it harder to see.

## Running Locally

Open `index.html` in a browser. No build tools or dependencies needed.

## Deploy

Hosted on [Netlify](https://happyfeetgame.netlify.app/).
